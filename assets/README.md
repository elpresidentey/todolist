# Assets Directory

## Background Image

Download a landscape mountain image from:
- **Unsplash**: https://unsplash.com/s/photos/mountain-landscape
- **Pexels**: https://www.pexels.com/search/mountain%20landscape/

### Recommended Images:
1. Purple/blue mountain landscapes
2. Landscape orientation (1920x1080 or similar)
3. Free to use without attribution

### Suggested Search Terms:
- "purple mountain landscape"
- "blue mountain sunset"
- "mountain gradient sky"

### File Name:
Save as: `mountain-bg.jpg` or `mountain-bg.png`

### Usage:
The image will be used as a background in the header section with a gradient overlay.
